### Important Keys in Coding

| Character | Name & Use (HTML/CSS Examples)                                               |
| --------- | ---------------------------------------------------------------------------- |
| `<`       | **[Less than](https://www.edclub.com/sportal/program-3/451.play)** ‚Äî starts HTML tags (`<div>`, `<p>`)                            |
| `>`       | **[Greater than](https://www.edclub.com/sportal/program-3/451.play)** ‚Äî ends HTML tags (`</div>`, `<p>` closes)                   |
| `/`       | **[Slash](https://www.edclub.com/sportal/program-3/179.play)** ‚Äî closes HTML tags (`</h1>`), used in self-closing tags (`<br />`) |
| `&`       | **[Ampersand](https://www.edclub.com/sportal/program-3/389.play)** ‚Äî writes HTML entities (`&amp;` for `&`, `&lt;` for `<`)       |
| `"`       | **[Double quote](https://www.edclub.com/sportal/program-3/433.play)** ‚Äî wraps HTML attribute values (`<a href="link.html">`)      |
| `'`       | **[Single quote](https://www.edclub.com/sportal/program-3/433.play)** ‚Äî alternative in attributes (`<a href='link.html'>`)        |
| `{` `}`   | **[Curly braces](https://www.edclub.com/sportal/program-3/445.play)** ‚Äî wrap CSS rule blocks (`h1 { color: red; }`)               |
| `:`       | **[Colon](https://www.edclub.com/sportal/program-3/236.play)** ‚Äî separates CSS property and value (`color: blue;`)                |
| `;`       | **[Semicolon](https://www.edclub.com/sportal/program-3/127.play)** ‚Äî ends CSS declarations (`font-size: 16px;`)                   |
| `(` `)`   | **[Parentheses](https://www.edclub.com/sportal/program-3/374.play)** ‚Äî used in CSS functions (`background: url('image.jpg')`)     |
| `[` `]`   | **[Square brackets](https://www.edclub.com/sportal/program-3/442.play)** - using in JavaScript for arrays (`names[0]`)
